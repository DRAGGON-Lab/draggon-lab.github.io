# V1 implementation plan

## Milestone PR sequence

1. Astro scaffold, Tailwind, TypeScript, folder structure, CI.
2. Content collections, schemas, validation, draft behavior.
3. Core pages and routing.
4. Seed real V1 content.
5. Design system, hero diagram, responsive layout, dark mode.
6. Deployment: GitHub Pages production, Cloudflare previews, Plausible, SEO, privacy.
7. README, docs, contributor files, roadmap, final launch polish.

## PR1 acceptance criteria

- pnpm install, pnpm dev, and pnpm build work.
- Tailwind and Astro configuration present.
- siteConfig.ts defines core identity.
- Homepage renders real DRAGGON Lab copy.
- Header/footer render the locked navigation and identity.
- CI runs format, lint, tests, preview content validation, and production build.
- README and docs explain the system.
