# Code of Conduct

DRAGGON Lab aims to be collaborative, rigorous, open, and constructive. We value careful thinking, respectful disagreement, clear documentation, interdisciplinary learning, and reusable work.

Contributors are expected to participate in a respectful and inclusive manner. Harassment, discrimination, personal attacks, or hostile behavior are not acceptable.

For concerns related to this repository or website, contact DRAGGON Lab at contact@draggonlab.org.
