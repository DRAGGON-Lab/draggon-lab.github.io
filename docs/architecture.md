# DRAGGON Lab website architecture

The website is a modular public map of the DRAGGON Lab ecosystem: research interests, software tools, workflows, datasets, publications, teaching resources, people, and project directions.

## Main navigation

- Research
- Tools
- Publications
- Lab Notes
- Teaching
- People
- Join / Collaborate

No separate About page is included in Version 1. Identity is distributed through the homepage, People page, Join / Collaborate page, footer, and Lab Notes.

## Homepage order

1. Hero
2. Research Vision + five Research Interests
3. Tool Ecosystem Preview
4. Applications / Impact
5. Featured Publications
6. Featured Lab Notes
7. Join / Collaborate

## Tool ecosystem

Tools are organized by Design, Build, Test, Learn, and Infrastructure. The Tools page title is Platform Ecosystem.

## Content collections

Planned collections: research, tools, publications, lab-notes, teaching-modules, teaching-resources, people, project-directions, workflows, and datasets.

All content supports draft. Production excludes drafts globally. Preview/development may show drafts with badges.

## Site graph

/site-graph.json exposes compact published metadata only. It does not expose full MDX bodies, long abstracts, or BibTeX.

Relationship vocabulary: supports, uses, describes, authored_by, related_to, cites, produces, teaches, enables.
